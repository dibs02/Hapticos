Hapticos

Privacy Policy

Hapticos is a browser extension that plays local feedback sounds when you click supported elements on web pages and shows a local interaction count in the extension popup.

What data Hapticos uses

Hapticos processes click interactions on supported http and https pages in order to trigger local feedback sounds.

Hapticos stores a local interaction count using the Chrome storage API so the popup can display how many supported interactions were triggered.

What Hapticos does not collect

  • Hapticos does not collect personally identifiable information.
  • Hapticos does not collect passwords, payment data, or health data.
  • Hapticos does not collect webpage text, form content, or messages.
  • Hapticos does not read or sync browser history.

How data is stored

The interaction count is stored locally in the browser using chrome.storage.local. Hapticos does not send this data to external servers.

Data sharing

Hapticos does not sell, transfer, or share user data with third parties.

Remote code

Hapticos does not use remote code. Its scripts and assets are bundled with the extension package.

Contact

If you have questions about this policy, contact the developer through the project repository: github.com/dibs02/Hapticos
